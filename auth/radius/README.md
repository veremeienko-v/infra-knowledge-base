# 📡 RADIUS

## 📌 Опис
Цей розділ містить документацію та налаштування RADIUS сервера для централізованої аутентифікації.

Охоплює:
- перевірку доступу користувачів  
- інтеграцію з Active Directory  
- контроль доступу до мережі (VPN, Wi-Fi)  

---

## 🧩 Склад

### NPS (Network Policy Server)
Реалізація RADIUS на базі Windows Server:
- політики доступу  
- правила аутентифікації  
- інтеграція з AD  

👉 [Перейти](auth/radius/nps)

---

## 📂 Структура

- docs/ → інструкції  
- configs/ → налаштування  
- policies/ → політики доступу  
- scripts/ → автоматизація  

---

## 🔐 Безпека

- використовувати сильні методи аутентифікації  
- обмежити доступ клієнтів RADIUS  
- вести логування підключень  
- регулярно перевіряти політики  

---

## 🔗 Інтеграції

- VPN (OpenVPN)  
- Wi-Fi (UniFi)  
- Active Directory  

---

## 📊 Практики

- використовувати групи AD для доступу  
- мінімізувати кількість правил  
- чітко розділяти політики  

---

## 📎 Примітка
RADIUS використовується для централізованого контролю доступу до мережевих ресурсів.
