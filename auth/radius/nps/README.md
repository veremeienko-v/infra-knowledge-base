# 🔑 NPS / RADIUS

## 📌 Опис
Налаштування Microsoft NPS як RADIUS сервера.

Охоплює:
- політики доступу
- інтеграцію з AD
- контроль підключень

---

## 📂 Вміст

- 📄 Налаштування
- 📜 Політики

---

## 📁 Структура

- docs/ → інструкції  
- configs/ → конфігурації  
- policies/ → правила  

---

## 🔐 Безпека

- використовувати групи доступу  
- обмежити клієнтів  
- вести логування  

---

## 📎 Примітка
NPS використовується для централізованої аутентифікації.
