# 🔑 Authentication

## 📌 Опис
Цей розділ містить документацію, конфігурації та скрипти для систем аутентифікації та контролю доступу.

Охоплює:
- централізовану аутентифікацію  
- контроль доступу до сервісів  
- інтеграцію з Active Directory  
- політики безпеки  

---

## 🧩 Склад

### RADIUS
Система централізованої аутентифікації:
- перевірка доступу  
- інтеграція з мережевими сервісами  
- контроль підключень  

👉 [Перейти](radius)

---

## 📂 Структура

Кожен сервіс має стандартну структуру:

- docs/ → документація  
- configs/ → конфігурації  
- scripts/ → автоматизація  
- policies/ → політики доступу  

---

## 🔐 Безпека

- використовувати принцип мінімальних прав (least privilege)  
- застосовувати групи доступу  
- вести аудит і логування  
- обмежувати доступ по IP та ролях  

---

## 📊 Практики

- централізувати аутентифікацію  
- уникати локальних акаунтів  
- використовувати інтеграцію з AD  
- регулярно перевіряти політики  

---

## 📎 Примітка
Authentication є критичною частиною безпеки всієї інфраструктури.
