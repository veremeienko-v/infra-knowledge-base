# В данній інструкції показаний короткий опис налаштування NPS в Windows Server 2025

## Завантажте файл [Налаштування NPS](configuration-nps-server-on-windows-server-2025.docx)
> Після проведення налаштувань, описаних в інструкції з налаштування NPS
- Створіть текстовий файл, для прикладу "C:\VPN\vpn_users.txt"
- Створіть групу в AD, наприклад: "vpn", користувачам якої, будуть створюваться файли конфігурацій та сертифікати OpenVPN сервера, у випадку додавання до групи, а у випадку видалення користувачів з групи, буде запускатись процес відгуку сертифікатів та видалення клієнтського файла конфіга на сервері OpenVPN/
- Створіть скрипт powershell [автостворення конфігів](../scripts/vpn-autosync.ps1), який буде запускати процес створення/відкуку сертифікатів на сервері OpenVPN
