# 📊 Monitoring

## 📌 Опис
Цей розділ містить документацію, конфігурації та скрипти для систем моніторингу інфраструктури.

Охоплює:
- моніторинг серверів та мережі  
- збір метрик  
- алерти та сповіщення  
- візуалізацію даних  

---

## 🧩 Склад

### Zabbix
Система моніторингу інфраструктури:
- сервер моніторингу  
- агенти  
- тригери та алерти  
- шаблони  

👉 Перейти

---

## 📂 Структура

Кожен сервіс має стандартну структуру:

- docs/ → документація  
- configs/ → конфігурації  
- scripts/ → автоматизація  
- templates/ → шаблони моніторингу  

---

## 🔐 Безпека

- обмежити доступ до системи моніторингу  
- використовувати HTTPS  
- контролювати доступ до агентів  
- налаштувати алерти на підозрілі події  

---

## 📊 Практики

- моніторити критичні сервіси  
- використовувати шаблони  
- налаштовувати алерти тільки для важливих подій  
- уникати “alert fatigue”  

---

## 📎 Примітка
Система моніторингу є ключовою для стабільності інфраструктури та швидкого реагування на інциденти.
