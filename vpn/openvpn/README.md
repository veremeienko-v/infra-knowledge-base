# 🔐 OpenVPN

## 📌 Опис
Налаштування та конфігурація OpenVPN для організації VPN.

Охоплює:
- сервер
- клієнтів
- сертифікати

---

## 📂 Вміст

- 📄 Встановлення
- 👥 Клієнти
- 🔐 Безпека

---

## 📁 Структура

- docs/ → інструкції  
- configs/ → .conf / .ovpn  
- scripts/ → автоматизація  
- clients/ → клієнтські конфіги  

---

## 🔐 Безпека

- використовувати сертифікати  
- обмежувати доступ  
- регулярно оновлювати ключі  

---

## 📎 Примітка
VPN є критичною частиною доступу до інфраструктури
