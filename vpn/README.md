# 🔐 VPN

## 📌 Опис
Цей розділ містить документацію, конфігурації та скрипти для налаштування VPN інфраструктури.

Охоплює:
- віддалений доступ користувачів
- site-to-site VPN
- шифрування трафіку
- контроль доступу

---

## 🧩 Склад

### OpenVPN
Розгортання та налаштування VPN сервера:
- серверна конфігурація
- клієнтські профілі
- сертифікати
- безпека

👉 [Перейти](openvpn)

---

## 📂 Структура

Кожен сервіс має стандартну структуру:

- docs/ → документація  
- configs/ → конфігурації  
- scripts/ → автоматизація 

---

## 🔐 Безпека

- використовувати сильні алгоритми шифрування  
- обмежувати доступ по IP або групах  
- регулярно оновлювати сертифікати  

---

## 📎 Примітка
VPN є критичною частиною безпеки — всі зміни повинні бути протестовані перед використанням.
