# 🌐 Network

## 📌 Опис
Цей розділ містить документацію, конфігурації та скрипти для мережевої інфраструктури.

Охоплює:
- маршрутизатори
- комутатори
- Wi-Fi інфраструктуру
- мережеву безпеку

---

## 🧩 Склад

### MikroTik
Конфігурації та налаштування RouterOS:
- firewall
- NAT
- VPN
- routing

👉 [Перейти](mikrotik)

---

### UniFi
Керування Wi-Fi інфраструктурою:
- контролер
- точки доступу
- мережеві політики

👉 Перейти

---

## 📂 Структура

Кожен сервіс має стандартну структуру:

- docs/ → документація  
- configs/ → конфігурації  
- scripts/ → автоматизація  
- backups/ → резервні копії  

---

## 🔐 Безпека

- всі firewall правила повинні бути задокументовані  
- доступ до мережевого обладнання обмежений  
- резервні копії конфігурацій повинні зберігатися окремо  

---

## 📎 Примітка
Перед внесенням змін у мережеву конфігурацію рекомендується перевіряти їх у тестовому середовищі.
