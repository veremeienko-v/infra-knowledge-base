# 🌐 UniFi

## 📌 Опис
Документація для UniFi Network Application та мережевої інфраструктури UniFi.

Охоплює:
- контролер
- точки доступу
- мережеві політики

## 📂 Вміст

- 📄 Встановлення
- ⚙️ Налаштування
- 💾 Бекапи

## 📁 Структура

- docs/ → інструкції  
- configs/ → конфігурації  
- scripts/ → автоматизація  
- backups/ → резервні копії

## 🔐 Безпека

- доступ до контролера обмежений  
- регулярні резервні копії  
- контроль доступу до Wi-Fi
