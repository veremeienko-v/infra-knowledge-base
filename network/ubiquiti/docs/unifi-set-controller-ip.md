## Щоб прописати IP контролера на точці доступу Ubiquiti UniFi через SSH:

1. Підключіться до точки (логін та пароль за замовчуванням - **ubnt**):
```bash
ssh ubnt@IP_ТОЧКИ
```
2. Виконайте команду:
```bash
set-inform http://IP_КОНТРОЛЕРА:8080/inform
```
Наприклад:
```bash
set-inform http://192.168.1.10:8080/inform
```
3. Після цього точка повинна з’явитися в контролері для Adopt.

ℹ️ Іноді команду потрібно виконати двічі:

- перший раз — для появи в контролері,
- другий — після натискання Adopt.

Перевірити статус:
```bash
mca-cli-op info
```
Або:
```bash
info
```
