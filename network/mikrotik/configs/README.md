# ⚙️ Configurations

## 📌 Опис
Цей розділ містить конфігураційні файли сервісу.

Охоплює:
- основні налаштування
- шаблони конфігів
- приклади

---

## 📂 Вміст

- *.conf → конфігурації  
- *.rsc → MikroTik  
- *.yaml → сервіси  
- *.json → налаштування  

---

## 📏 Стандарти

- кожен файл має опис  
- зрозумілі назви  
- логічна структура  

---

## ⚠️ Важливо

- не зберігати секрети  
- використовувати шаблони  
- документувати зміни  

---

## 🔐 Безпека

- видаляти паролі перед комітом  
- використовувати placeholders  
- контролювати доступ  

---

## 📎 Примітка
Конфігурації повинні відповідати документації та бути готовими до використання.
