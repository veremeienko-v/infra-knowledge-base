# 🧩 SBC (Single Board Computers)

## 📌 Опис
Цей розділ містить документацію, конфігурації та скрипти для одноплатних комп’ютерів (SBC).

Охоплює:
- налаштування пристроїв  
- оптимізацію системи  
- використання в інфраструктурі  
- лабораторні та тестові середовища  

---

## 🧩 Склад

### Raspberry Pi
Найпоширеніша платформа SBC:
- базова конфігурація  
- мережеві налаштування  
- використання як сервер  

👉 [Перейти](raspberry-pi)

---

## 📂 Структура

Кожна платформа має стандартну структуру:

- docs/ → документація  
- configs/ → конфігурації  
- scripts/ → автоматизація  

---

## 🔐 Безпека

- змінити стандартні паролі  
- обмежити доступ по SSH  
- використовувати ключі замість паролів  
- регулярно оновлювати систему  

---

## ⚙️ Практики

- використовувати SBC для тестових середовищ  
- розгортати легкі сервіси  
- контролювати використання ресурсів  
- застосовувати моніторинг  

---

## 📎 Примітка
SBC часто використовуються для лабораторій, прототипів та невеликих сервісів в інфраструктурі.
