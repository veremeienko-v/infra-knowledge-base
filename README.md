# 🧠 База знань з інфраструктури та системного адміністрування

## 📌 Опис
Цей репозиторій містить документацію, конфігурації та скрипти, пов’язані з налаштуванням інфраструктури та системним адмініструванням.

Охоплює:
- 🌐 Мережеве обладнання (MikroTik, UniFi)
- 🔐 VPN рішення (OpenVPN, WireGuard)
- 📊 Системи моніторингу (Zabbix)
- 🐧 Налаштування та захист Linux серверів
- 🪟 Налаштування Windows Server (NPS, Active Directory)
- ⚙️ Скрипти автоматизації та резервне копіювання

---

## 🎯 Мета
- Централізувати всі знання по інфраструктурі
- Забезпечити відтворюваність налаштувань
- Зберігати конфігурації та скрипти
- Використовувати як особисту або командну базу знань

---

## 🧩 Структура

- network/ → мережа (MikroTik, UniFi)
- vpn/ → VPN сервіси
- monitoring/ → системи моніторингу
- linux/ → Linux системи
- windows/ → Windows Server
- auth/ → аутентифікація (RADIUS, NPS)
- directory/ → Active Directory
- integrations/ → інтеграції сервісів
- scripts/ → автоматизація
- configs/ → конфігурації
- backups/ → резервні копії
- docs/ → загальна документація
